# VETGUIA

## Descripción

VETGUIA es una aplicación web desarrollada para brindar orientación sobre el cuidado de mascotas, primeros auxilios veterinarios y atención en situaciones de emergencia.

El sistema reúne información práctica y organizada para apoyar a los propietarios de mascotas en la toma de decisiones ante incidentes comunes, así como en la consulta de material educativo relacionado con la salud y el bienestar animal.

---

## Objetivos

* Facilitar el acceso a información básica sobre emergencias veterinarias.
* Promover buenas prácticas en el cuidado de mascotas.
* Centralizar guías, recomendaciones y material educativo en una única plataforma.
* Ofrecer una interfaz intuitiva y accesible para los usuarios.

---

## Funcionalidades

* Consulta de guías de emergencia.
* Acceso a material educativo.
* Información sobre primeros auxilios para mascotas.
* Visualización de folletos y recursos informativos.
* Gestión de usuarios mediante inicio de sesión y registro.
* Navegación adaptada a dispositivos móviles y de escritorio.

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/dlopez695/Proyecto-VetGuia.git
```

Ingresar al directorio del proyecto:

```bash
cd Proyecto-VetGuia
```

Instalar dependencias:

```bash
npm install
```

Ejecutar en modo desarrollo:

```bash
npm run dev
```

La aplicación estará disponible en:

```text
http://localhost:5173
```

---

## Compilación para Producción

```bash
npm run build
```

Los archivos optimizados se generarán en la carpeta:

```text
dist/
```
